Tree manipulation & utilities
=============================

- :doc:`Branch length multiplier <../commands/branch_length_multiplier>` - Multiply branch lengths by a factor
- :doc:`Chronogram <../commands/chronogram>` - Time-calibrated tree with geological timescale (rectangular or circular)
- :doc:`Collapse bipartitions <../commands/collapse_branches>` - Collapse low-support bipartitions
- :doc:`Internode labeler <../commands/internode_labeler>` - Label internal nodes of a tree
- :doc:`Last common ancestor subtree <../commands/last_common_ancestor_subtree>` - Extract subtree from LCA of specified taxa
- :doc:`Monophyly check <../commands/monophyly_check>` - Test monophyly of a group of taxa
- :doc:`Nearest neighbor interchange <../commands/nearest_neighbor_interchange>` - Generate NNI tree rearrangements
- :doc:`Print tree <../commands/print_tree>` - Print ASCII representation of a tree
- :doc:`Prune tree <../commands/prune_tree>` - Prune taxa from a tree
- :doc:`Rename tree tips <../commands/rename_tree_tips>` - Rename tip labels in a tree
- :doc:`Root tree <../commands/root_tree>` - Root or reroot a tree
- :doc:`Subtree pruning and regrafting <../commands/subtree_prune_regraft>` - Generate all SPR rearrangements for a specified subtree
- :doc:`Tip labels <../commands/tip_labels>` - Print tip labels of a tree
- :doc:`Transfer annotations <../commands/transfer_annotations>` - Transfer node annotations between trees (e.g., wASTRAL to RAxML/IQ-TREE)
