Tree comparison & consensus
===========================

- :doc:`Consensus network <../commands/consensus_network>` - Consensus network from multiple trees
- :doc:`Consensus tree <../commands/consensus_tree>` - Consensus tree from multiple trees
- :doc:`Cophylogenetic plot (tanglegram) <../commands/cophylo>` - Tanglegram for comparing two trees
- :doc:`Evolutionary tempo mapping <../commands/evo_tempo_map>` - Detect rate-topology associations in gene trees
- :doc:`Kuhner-Felsenstein distance <../commands/kf_distance>` - Branch score distance between trees (topology + branch lengths)
- :doc:`NeighborNet <../commands/neighbor_net>` - NeighborNet phylogenetic network from distance matrix (Bryant & Moulton 2004)
- :doc:`Polytomy testing <../commands/polytomy_test>` - Test for polytomies in a tree
- :doc:`Quartet network <../commands/quartet_network>` - Quartet-based network visualization
- :doc:`Quartet pie chart <../commands/quartet_pie>` - Phylogram with quartet concordance pie charts at internal nodes
- :doc:`Robinson-Foulds distance <../commands/robinson_foulds_distance>` - Topological distance between trees
- :doc:`Spectral discordance decomposition <../commands/spectral_discordance>` - PCA ordination and clustering of gene tree topologies
- :doc:`Tree space visualization <../commands/tree_space>` - Visualize gene tree topology space via MDS, t-SNE, UMAP, or clustered distance heatmap
