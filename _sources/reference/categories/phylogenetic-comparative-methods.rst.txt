Phylogenetic comparative methods
================================

- :doc:`Phylo GWAS <../commands/phylo_gwas>` - Phylogenetic genome-wide association study
- :doc:`Phylogenetic ANOVA / MANOVA <../commands/phylo_anova>` - Phylogenetic ANOVA or MANOVA using RRPP (Adams & Collyer 2018)
- :doc:`Phylogenetic GLM <../commands/phylogenetic_glm>` - Phylogenetic generalized linear model (supports discordance-aware VCV with ``-g``)
- :doc:`Phylogenetic Logistic Regression <../commands/phylo_logistic>` - Phylogenetic logistic regression for binary traits (Ives & Garland 2010)
- :doc:`Phylogenetic Ordination <../commands/phylogenetic_ordination>` - Ordination incorporating phylogenetic structure (supports discordance-aware VCV with ``-g``)
- :doc:`Phylogenetic path analysis <../commands/phylo_path>` - Compare causal DAGs via d-separation + PGLS (von Hardenberg & Gonzalez-Voyer 2013)
- :doc:`Phylogenetic regression (PGLS) <../commands/phylogenetic_regression>` - Phylogenetic generalized least squares regression (supports discordance-aware VCV with ``-g``)
- :doc:`Phylomorphospace <../commands/phylomorphospace>` - Phylomorphospace visualization
