Trait evolution
===============

- :doc:`Ancestral state reconstruction <../commands/ancestral_state_reconstruction>` - Reconstruct ancestral character states
- :doc:`Character map (synapomorphy/homoplasy mapping) <../commands/character_map>` - Map synapomorphies and homoplasies onto a phylogeny
- :doc:`Concordance-aware ancestral state reconstruction <../commands/concordance_asr>` - ASR incorporating gene tree discordance
- :doc:`Continuous trait evolution model comparison (fitContinuous) <../commands/fit_continuous>` - Compare continuous trait evolution models (supports discordance-aware VCV with ``-g``)
- :doc:`Continuous trait mapping (contMap) <../commands/cont_map>` - Map continuous traits onto a phylogeny
- :doc:`Density map <../commands/density_map>` - Posterior density of stochastic character maps
- :doc:`Discrete trait evolution model comparison (fitDiscrete) <../commands/fit_discrete>` - Compare ER, SYM, ARD Mk models
- :doc:`Disparity through time (DTT) <../commands/dtt>` - Disparity-through-time analysis with MDI statistic (Harmon et al. 2003)
- :doc:`Independent contrasts (PIC) <../commands/independent_contrasts>` - Felsenstein's phylogenetically independent contrasts
- :doc:`Multi-regime OU models (OUwie) <../commands/ouwie>` - Multi-regime Ornstein-Uhlenbeck models
- :doc:`OU shift detection (l1ou) <../commands/ou_shift_detection>` - Detect OU regime shifts on a phylogeny
- :doc:`Parsimony score <../commands/parsimony_score>` - Fitch parsimony score of a tree given an alignment
- :doc:`Phenogram (traitgram) <../commands/phenogram>` - Phenogram visualizing trait evolution
- :doc:`Phylogenetic heatmap <../commands/phylo_heatmap>` - Phylogeny alongside a heatmap of numeric trait values
- :doc:`Phylogenetic imputation <../commands/phylo_impute>` - Impute missing trait values using phylogenetic relationships
- :doc:`Rate heterogeneity test (multi-rate Brownian motion) <../commands/rate_heterogeneity>` - Test for rate heterogeneity in trait evolution
- :doc:`SIMMAP summary <../commands/simmap_summary>` - Per-branch SIMMAP summary with node posteriors (describe.simmap)
- :doc:`Stochastic character mapping (SIMMAP) <../commands/stochastic_character_map>` - Stochastic character mapping on a phylogeny
- :doc:`Threshold model <../commands/threshold_model>` - Felsenstein threshold model for trait correlation
- :doc:`Trait rate map <../commands/trait_rate_map>` - Per-branch evolutionary rate map for a continuous trait
