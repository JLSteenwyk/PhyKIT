Introgression & gene flow
=========================

- :doc:`D-statistic (ABBA-BABA test) <../commands/dstatistic>` - Patterson's D-statistic for detecting introgression
- :doc:`DFOIL test (Pease & Hahn 2015) <../commands/dfoil>` - DFOIL test for detecting and polarizing introgression in a 5-taxon symmetric phylogeny
- :doc:`Discordance asymmetry <../commands/discordance_asymmetry>` - Test for asymmetric discordance (gene flow detection)
- :doc:`Hybridization analysis <../commands/hybridization>` - Estimate reticulation events and localize hybridization on a species tree
