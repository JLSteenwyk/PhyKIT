How should character changes be displayed on branches?

Compare these approaches against the reference figure (filled = synapomorphy, open = homoplasy)

A: Classic Winclada Style

Filled circles = synapomorphies, open circles = homoplasies. Character number above, state transition below. Matches the reference figure exactly.

B: Simplified (New State Only)

Same circle convention but shows only the derived state below (not the full transition). Cleaner when branches have many changes. This is what the reference figure actually does.

C: Color-Coded (PhyKIT Style)

Uses color instead of fill/outline to distinguish synapomorphies (blue) from homoplasies (red/gray). More modern look, but departs from the classic convention. Could optionally distinguish convergence vs reversal.
