Full Character Map Layout

Cladogram with color-coded character changes on branches

character_map output — cladogram mode (default)

Key Features

  • Blue circles — Synapomorphies (unique changes supporting a clade)
  • Red circles — Convergences (same state gained independently on multiple branches)
  • Gray circles — Reversals (return to a previously observed state)
  • Character number above each circle, state transition below
  • Summary statistics printed below the figure
  • Colors customizable via --colors "blue,red,gray"
